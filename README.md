--README

1. Descargue los elementos que se encuentran adjuntos en ProyectoCoder en una carpeta de su equipo local.
2. Inicie Visual Studio Code, en caso tal que no cuente con la herramienta descargue el aplicativo e instale las diferentes herramientas (Python, Git, Django).
3. Cargue en Visual Studio Code la carpeta que contiene los archivos descargados y fue creada en su equipo. Encontrará las carpeta AppCoder, ProyectoCoder, Base de datos y manage.py.
4. Para conocer el contenido del proyecto ejecute una nueva terminar el Visual, dentro de la terminal dirijase a la carpeta donde se encuentra ubicado manage.py y ejecute la siguiente sentencia python manage.py runserver (Windows) python3 manage.py runserver (IOS)
5. Una vez generada la URL hacer click para poder observar el contenido de la página.  
6. En la página va a observar las pestañas correspondientes a los Curso, Profesores, Acudientes, Horarios, Materia y Estudiantes, donde podrá insertar la información en cada una de ellas y podrá realizar la busqueda de algun elemento en la pestaña Curso.
